# PALIS 九类 OC 设定卡编辑器

直接双击 `palis-oc-card-editor.html`，或从 Codex 中点击该文件，即可在浏览器打开，不需要安装软件。

## 如何编辑

1. 从左侧选择九种档案卡之一：国家、组织、科考站点、白幕入口、生态、相关人物、事件、异常、物种。
2. 在右侧修改字段，卡片会实时更新；也可以点击卡面上的文字，自动定位到对应输入框。
3. 修改会自动保存在当前浏览器中。关闭页面后再次打开，内容仍会保留。
4. “导出 PNG”用于发布图片；“导出 SVG”可在 Illustrator、Figma、Inkscape 等矢量软件中继续排版。
5. “导出整套数据”会保存九张卡的 JSON 备份。更换电脑或浏览器后，用“导入数据”恢复。

## 当前示例

九张卡都使用 P31 托马斯·E·马洛里的现有档案作为示例，但每张卡的字段和构图都不同。你可以把角色姓名和全部字段替换成自己的 OC，不会改动网页项目中的原始档案。
